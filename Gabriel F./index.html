Projeto SAGA SENAI: Big Street

O Problema: A desorganização, filas e atritos na hora de usar quadras esportivas (públicas e privadas).

A Solução: O Grupo Magid desenvolveu uma plataforma completa de gestão esportiva, com fila virtual e agendamento inteligente. Para tornar a experiência imersiva, implementamos um sistema de avaliação das quadras, consumimos uma API de localização para encontrar os espaços mais próximos e criamos um chat próprio para a comunicação entre os jogadores.

Tecnologias utilizadas: HTML, CSS, JavaScript e Python.
